# HPC Neura Execution Engine

The HPC Neura Execution Engine is a powerful workflow execution system designed to process complex AI-driven workflows with high performance and reliability. It provides a flexible, scalable platform for executing neural computation graphs integrated with various AI models and blockchain operations.

<Callout type="info">
The HPC Neura Execution Engine powers the core workflow execution capabilities of the NeuraLabs platform, enabling seamless integration of AI models, blockchain operations, and custom logic.
</Callout>

## Key Features

<Features>
  <Feature title="Flow-Based Execution" icon="network">
    Visual workflow design with drag-and-drop elements connected in a directed graph
  </Feature>
  <Feature title="Multi-Model AI Support" icon="ai-workflow">
    Integration with multiple AI providers including AWS Bedrock, Anthropic, and DeepSeek
  </Feature>
  <Feature title="Blockchain Integration" icon="blockchain">
    Native support for SUI blockchain operations and smart contract interactions
  </Feature>
  <Feature title="Real-Time Streaming" icon="terminal">
    Server-sent events (SSE) for streaming AI responses and execution updates
  </Feature>
</Features>

## Architecture Overview

The execution engine consists of:

- **Flow Executor**: Core engine that processes workflow graphs
- **Element Registry**: Extensible system of workflow elements
- **Service Layer**: Integration with external services and APIs
- **WebSocket API**: Real-time communication with clients
