## Why NeuraLabs?

NeuraLabs addresses key challenges in the AI landscape:

- **<img src="/img/icons/error.svg" width="16" height="16" style={{ verticalAlign: 'middle', marginRight: '4px' }} />Centralized Control** - Major platforms gatekeep AI access
- **<img src="/img/icons/warning.svg" width="16" height="16" style={{ verticalAlign: 'middle', marginRight: '4px' }} />Privacy Concerns** - User data often exploited
- **<img src="/img/icons/info.svg" width="16" height="16" style={{ verticalAlign: 'middle', marginRight: '4px' }} />Limited Monetization** - Creators can't easily profit from AI tools
- **<img src="/img/icons/success.svg" width="16" height="16" style={{ verticalAlign: 'middle', marginRight: '4px' }} />Our Solution** - Decentralized, privacy-preserving AI platform
