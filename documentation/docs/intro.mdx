# Welcome to NeuraLabs Documentation

NeuraLabs is a decentralized AI workflow platform built on the SUI blockchain, designed to democratize access to AI capabilities through a visual workflow builder and NFT-based access control system.

## Prerequisites

Before diving into the implementation guides, ensure you have:

```bash
# Required tools
- Node.js 18+ and npm/yarn
- Python 3.12 with conda
- SUI CLI tools
- Docker and Docker Compose
- Basic understanding of blockchain concepts
```

## Getting Started

### Quick Start

```bash
# Clone the repository
git clone https://github.com/neuralabs/neuralabs-sui
cd neuralabs-sui

# Install dependencies
npm install
cd frontend && npm install
cd ../neuralabs-backend && pip install -r requirements.txt

# Start services
docker-compose up -d
```

For detailed setup instructions, follow our [Getting Started Guide](./theoretical/getting-started).
