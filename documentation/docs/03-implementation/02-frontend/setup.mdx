## Overview

The NeuraLabs frontend is built with:
- React 18
- TypeScript
- Tailwind CSS
- React Flow for the visual builder
- SUI TypeScript SDK for blockchain integration

## Quick Start

```bash
cd frontend
npm install
npm start
```

For more details, see the [main setup guide](/docs/theoretical/getting-started).
