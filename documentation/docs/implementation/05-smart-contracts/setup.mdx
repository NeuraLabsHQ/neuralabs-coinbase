## Overview

NeuraLabs smart contracts include:
- NFT-based access control
- Workflow ownership management
- SUI Seal integration for encryption
- Decentralized storage integration

## Quick Start

```bash
cd mov_contract
sui move build
sui move test
sui client publish --gas-budget 100000000
```

For deployment details, see the [deployment guide](./deployment-guide.md) in the mov_contract folder.
