## Overview

The HPC Execution Node handles:
- AI workflow execution
- Multi-model support (Anthropic, DeepSeek, AWS Bedrock)
- Custom code execution blocks
- Streaming responses
- Workflow orchestration

## Quick Start

```bash
cd hpc-execution-node-backend/code_executor
pip install -r requirements.txt
python app.py
```

The execution API will be available at `http://localhost:8001`

For more details, see the [main setup guide](/docs/theoretical/getting-started).
