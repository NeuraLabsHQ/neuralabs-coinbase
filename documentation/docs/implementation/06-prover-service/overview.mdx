## Overview

The Prover Service handles:
- Zero-knowledge login (zkLogin) implementation
- Proof generation for privacy-preserving authentication
- Integration with SUI's zkLogin system
- OAuth provider integration

## Architecture

The prover service is integrated within the main backend service and provides:
- JWT token generation with ZK proofs
- Session management
- Privacy-preserving authentication flow

For more details, see the [main setup guide](/docs/theoretical/getting-started).
